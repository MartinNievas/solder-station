# Solder Station

En este repositorio se encuentran los códigos y circuitos para fabricar una estación de soldado.

## Requerimientos

### Hardware
  - Arduino pro-mini
  - Punta de soldador Hakko 207 (repuesto) 
  - Fuente 12V@10A
### Software
  - Para compilar y cargar al arduino [platformio](http://platformio.org/).
